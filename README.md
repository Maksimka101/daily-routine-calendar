# Daily Routine Calendar

Это адаптивный календарь, который пересчитывает график в зависимости от времени сна, подсказывая точные «окна» для кофе, тренировок и еды. Избавляет от ручных расчетов, помогает выжать максимум из дня и качественно восстановлениться ночью. Просто выстави время отбоя — и получи готовую инструкцию, как провести день бодро.
